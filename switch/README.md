# switch
![switch](switch.gif)

## 概要
switchは、式の値に応じて分岐処理を行うための構文です。caseによって条件となる複数の値や変数を指定できます。Swiftでは、switchのcase文から抜けるためのbreakを書かなくても良いことや、Double型・enum型・Classを条件に指定できること、複数の条件が列挙できることが特徴です。
