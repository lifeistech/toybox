# if
![if](if.gif)

## 概要
ifは、条件式の値に応じて分岐処理を行うための構文です。`&&,カンマ,||`でいくつかの条件式を結ぶことで複数条件の式を書くことができます。また、Bool型の変数を使った条件式で分岐処理をするときは`== true`を省略することができます。
