## 概要
UISegmentedControlとは何か
水平方向に配置された複数のボタンの集まりで各セグメントは個別のボタンとして機能する。いくつかの選択肢の中から1つだけ選ばせたい時に使えるコントロール。タイトル又は画像を表示することができます。

## どう使われているか
- AppStoreの購入済みApp一覧の[すべて]と[このiPhone上にない]の切り替え
